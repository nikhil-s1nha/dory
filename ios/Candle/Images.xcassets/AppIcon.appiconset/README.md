# App Icon Placeholder

This directory contains the app icon asset catalog for iOS.

## Required Icon Sizes

- 20x20 @2x, @3x (40x40, 60x60) - Notification icons
- 29x29 @2x, @3x (58x58, 87x87) - Settings icons
- 40x40 @2x, @3x (80x80, 120x120) - Spotlight icons
- 60x60 @2x, @3x (120x120, 180x180) - App icons
- 1024x1024 - App Store icon

Replace the placeholder entries in `Contents.json` with actual icon images, or add the icon images to this directory and update `Contents.json` accordingly.
